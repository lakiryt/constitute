# 解説：天皇の地位

この条文は、昭和憲法第一条を踏襲したものである。

> 〔昭和憲法 第一条〕 天皇は、日本国の象徴であり日本国民統合の象徴であつて、この地位は、主権の存する日本国民の総意に基く。

昭和憲法第一条は、象徴としての天皇の地位のみならず、国民主権をも定めていると解されてきた。

確かに同条が国民主権を前提としていることは明らかであるが、文章の構造からして、国民主権の積極的な宣言と捉えるには無理がある。
また、明治憲法と対比すれば同条の国民主権の主張はなお顕著となるが、憲法典の外の資料によってはじめて適切な解釈が可能となる表現は、広く国民が合意を以てその統治を委任すべく制定する文書には馴染まない。

もっとも、同条および前文のほかに国民主権を明示する箇所が存在しない昭和憲法の構造を前提とすれば、このような無理のある解釈をせざるを得ないことは理解できる。

この憲法においては国民主権を定めた条文は別にあるため、この条文は、天皇の地位とその根拠を定めるにとどまるものと字義通りの解釈をして差し支えない。

## 踏襲する論点

### 天皇の民事裁判権
象徴としての地位を理由に、天皇の民事裁判権を否定した判例（[最高裁平成元年11月20日第二小法廷判決](https://www.courts.go.jp/assets/hanrei/hanrei-pdf-52451.pdf)）がある。

昭和憲法第一条を踏襲により、この憲法は、この論点も踏襲することとなる。

### 天皇の人権
天皇や皇族は参政権、婚姻、職業選択の自由など一部の人権について、皇族でない国民とは異なる形で制約を受けてきた。

象徴天皇制の踏襲により、この憲法は、この論点をも踏襲することとなる。
